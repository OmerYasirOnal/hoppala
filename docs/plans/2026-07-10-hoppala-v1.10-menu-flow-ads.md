# Hoppala v1.10 — Sky Poster menu, coherent flow, honest ads — Implementation Plan

> **For agentic workers:** implement task-by-task with TDD + frequent commits. Steps use checkbox syntax.

**Goal:** Redesign the bare menu into a "Sky Poster" entry screen, make screen transitions coherent, add the
missing game-over → main-menu path, and gate the rewarded-revive button on real ad-readiness.

**Architecture:** UI-adapter + platform-bridge only. `screens.ts` gains a poster-card layout + a transition
helper; `styles.css` gains the poster/mascot/animation styles; `main.ts` gates the revive on ad-readiness and
adds a game-over → menu handler; the `platform` bridge exposes ad readiness/preload. A pure `revive-policy.ts`
holds the gating decision, unit-tested.

**Tech Stack:** Vanilla TS, Canvas 2D, CSS, Vitest. `@capacitor-community/admob` (native, lazy chunk).

## Global Constraints (verbatim from spec)

- No `src/game/*` or `src/core/*` change — sim/spawner/RNG/determinism/reachability/daily-fairness byte-identical.
- Core bundle < 60KB gzip; AdMob stays in the lazy Capacitor chunk.
- UI copy in the `STRINGS` set (`src/ui/screens.ts`), auto TR/EN; reuse existing keys (`mainMenu`, `hint`, `play`).
- Transitions CSS-only (no JS anim loop), honor `prefers-reduced-motion`, hold 60fps.
- Interactive layers under `#ui` (pointer-events:none) must re-assert `pointer-events:auto`.
- Conventional commits; `pnpm test` + `pnpm build` stay green.

### Task 1: Pure revive-gating policy (`revive-policy.ts`)

**Files:** Create `src/ui/revive-policy.ts`, Test `src/ui/revive-policy.test.ts`.

**Produces:** `shouldOfferRevive(mode: GameMode, revivesUsed: number, hasNativeAd: boolean, nativeAdReady: boolean): boolean`

- [ ] Test (truth table): free+web(noNative)→true; free+native+ready→true; free+native+notReady→false; daily→false (both native states); revivesUsed≥1→false.
- [ ] Implement:
```ts
import type { GameMode } from '../platform/bridge';
/** Whether game-over should offer the rewarded revive. Pure → unit-tested without DOM/native. */
export function shouldOfferRevive(
  mode: GameMode, revivesUsed: number, hasNativeAd: boolean, nativeAdReady: boolean,
): boolean {
  if (mode !== 'free') return false;      // daily stays fair/deterministic
  if (revivesUsed >= 1) return false;     // one revive per run
  return hasNativeAd ? nativeAdReady : true; // web stub always available
}
```

let loading = false;
function preload(): void {
  if (loading) return;
  loading = true; loaded = false;
  AdMob.prepareRewardVideoAd(AD_OPTS)
    .then(() => { loaded = true; })
    .catch(() => { loaded = false; })
    .finally(() => { loading = false; });
